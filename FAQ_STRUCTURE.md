## 1. API и интеграция

### 1.1 Создание заказа (253 обращения)
**Частые вопросы:**
- Как создать многоместный заказ?
- Какие коды тарифов указывать?
- Как передать стоимость доставки (deliverySum)?
- Создание заказов для разных СД (5Post, Dalli, ПЭК)

**Ключевые поля:**
- `payment.deliverySum` - стоимость доставки
- `places[]` - массив мест для многоместных
- `place.barcode` - обязательное поле

### 1.2 Статусы заказа (500 обращений)
**Частые вопросы:**
- Какой статус у выкупленного/невыкупленного товара?
- Как получить актуальные данные о заказе?
- GET /v2/orders/{id}/details - структура ответа

### 1.3 Получение офферов (97 обращений)
**Частые вопросы:**
- Как получить срок доставки по конкретному заказу?
- Особенности офферов для Почты (тариф 23)
- Настройка Яндекс Доставки для офферов

### 1.4 Авторизация (79 обращений)
**Типичные ошибки:**
- 401 Unauthorized - неверный токен

- Истёк срок действия токена

### 1.5 Партии (77 обращений)
**Частые вопросы:**
- Как формировать партии Почты из ЕКОМ?

### 1.6 Редактирование заказа (47 обращений)
**Ограничения:**

**Настройка:**

### 7.1 Timeout/5xx (86 обращений)
**Типичные:**
- 500 Internal Server Error
- 502 Bad Gateway
- 503 Service Unavailable
- 504 Gateway Timeout

### 7.2 Валидация (43 обращения)
**Типичные:**
- Недопустимое значение поля

- Обязательное поле не заполнено

### 7.3 Интервал доставки (27 обращений)
**Типичные:**
- Некорректный интервал
- Интервал недоступен для выбранной даты

### 7.4 ПВЗ не найден (4 обращения)

## Рекомендации для AI-агента

1. **Приоритет контекста:**
   - Сначала определить категорию вопроса
   - Проверить готовые Q&A по категории
   - Использовать документацию API

2. **Ключевые regex-паттерны для классификации:**
   - Создание заказа: `/создан.*заказ|order.*creat|POST.*orders/i`
   - Статусы: `/статус.*заказ|GET.*orders|tracking/i`
   - Ошибки: `/500|502|503|timeout|error/i`
   - СД: `/сдек|cdek|5post|почта.*росси|dalli/i`

3. **Эскалация на оператора:**
   - Нет Q&A по теме
   - Технические ошибки инфраструктуры
   - Финансовые вопросы
   - Индивидуальные настройки клиента
